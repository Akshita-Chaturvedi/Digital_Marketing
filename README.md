# 📊 Online Advertising Performance Analysis

This project analyzes real-world online advertising campaign data to evaluate performance and identify optimization opportunities.

## 🚀 Objective
To understand how different factors like banner size, placement, and campaign strategy affect:
- Click-through rate (CTR)
- Conversion rate
- Cost efficiency (CPC)
- Return on Investment (ROI)

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📈 Key Analysis Performed
- Trend analysis of displays and clicks over time
- Banner size vs CTR comparison
- Placement vs conversion rate analysis
- Cost vs revenue relationship
- Campaign-wise ROI comparison
- Outlier detection

---

## 🔍 Key Insights
- Certain banner sizes (e.g., 580x400) achieved higher CTR
- Specific placements showed significantly higher conversion rates
- Higher cost generally led to higher revenue, but not always better ROI
- Some campaigns outperformed others consistently in profitability

---

## 📊 Visualizations
The project includes multiple visualizations such as:
- Line plots (trend analysis)
- Bar charts (comparisons)
- Scatter plots (correlation)
- Box plots (outlier detection)

---


## 💡 Conclusion
This analysis demonstrates how data-driven insights can help optimize advertising strategies and improve ROI.
